# ☕ Cafe Web Sitesi

Bu proje, modern ve responsive bir **kafe tanıtım web sitesi** tasarımıdır. Sadece frontend teknolojileri kullanılarak geliştirilmiştir.

## 🚀 Proje Amacı

Bu web sitesinin amacı bir kafenin dijital ortamda tanıtımını yapmak ve kullanıcıların:
- Menüleri görüntülemesini
- Kafe hakkında bilgi almasını
- İletişim bölümüne ulaşmasını
- Görsel olarak modern bir arayüz deneyimlemesini

sağlamaktır.

## 🛠️ Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (varsa)
- Responsive Web Design

## ✨ Özellikler

- 📱 Responsive (mobil uyumlu) tasarım
- 🍽️ Menü bölümü
- 📸 Görsel galeri
- 📞 İletişim bölümü
- 🎨 Modern ve sade UI tasarım
